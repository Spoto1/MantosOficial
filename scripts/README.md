# Scripts

## Pastas

- `scripts/qa/`: scripts reutilizáveis de QA e smoke.
- `scripts/local/`: utilitários locais de conveniência.
- `scripts/archive/legacy/`: scripts antigos preservados apenas para referência.

## Comandos principais

- `npm run local:open`
- `npm run qa:admin:browser`
- `npm run qa:storefront:browser`
- `npm run qa:storefront:http`
