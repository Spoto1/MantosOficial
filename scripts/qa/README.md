# QA Scripts

## Entradas atuais

- `admin-browser-qa.mjs`: smoke browser do admin.
- `storefront-browser-qa.mjs`: smoke browser do storefront.
- `storefront-http-qa.ts`: smoke HTTP/server-side do storefront.

## Comandos

- `npm run qa:admin:browser`
- `npm run qa:storefront:browser`
- `npm run qa:storefront:http`

## Saídas

- screenshots e artefatos persistentes: `output/`
- logs temporários e scratch local: `tmp/`
